# CryptoTracker

It was created with Mvvm design pattern and Clean arc approach.


- Mvvm desing pattern
- Clean Arc.
- Architecture Components
  * WorkManager
  * Room
  * Livedata
  * Navigation
- Reactivex Coroutines&Flow
- RestFull api Retrofit
- DI Dagger&Hilt
- TestMock Mockito
- UnitTestReport Jacoco
- LintCheck Ktlint
